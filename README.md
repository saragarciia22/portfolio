# Portfolio de Sara García

Este es mi portfolio personal donde muestro mis proyectos, habilidades y experiencia en diseño gráfico y desarrollo web.

## Estructura del Proyecto

- `index.html`: Página principal del portfolio.
- `css/`: Estilos CSS.
- `js/`: Scripts JavaScript.
- `assets/`: Recursos estáticos.
  - `img/`: Imágenes y gráficos.
  - `docs/`: Documentos y certificados.
  - `logos/`: Logos y marcas.
  - `projects/`: Archivos de proyectos.
  - `certificates/`: Certificados.
- `fonts/`: Fuentes tipográficas.
- `docs/`: Documentación adicional.

## Cómo ejecutar

Abre `index.html` en tu navegador web.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
